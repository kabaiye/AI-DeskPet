# 我用 Electron 做了一个有灵魂的 AI 桌宠！开源免费，手把手教你拥有自己的桌面伙伴

## 前言

你有没有想过，桌面上趴着一只会说话、有情绪、还会主动关心你的小猫咪是什么体验？

不是那种循环播放动画的传统桌宠——而是一只接入了大模型、拥有独立人格、能看懂你屏幕内容、还会在你久坐时催你喝水的**AI 桌宠**。

今天给大家介绍我的开源项目 **AI-DeskPet**，默认角色是大家熟悉的**罗小黑**。完全免费、开源、可自定义角色，下面我会从功能展示到部署教学一步步带你上手。

---

## 一、这只桌宠能干什么？

### 1. AI 智能聊天

接入智谱 GLM 大模型，支持多轮对话，像和一个真正有性格的朋友聊天。小黑会根据你设置的性格模式（活泼可爱/温柔体贴/调皮捣蛋/高冷傲娇）用完全不同的语气回复你。

> 【图片占位：聊天窗口截图，展示一段多轮对话，小黑用可爱语气回复】

### 2. 截图问答

按下快捷键 `Alt+F2`，框选屏幕任意区域，AI 会分析截图内容并回答你的问题。看到代码帮你解释，看到美食跟你一起馋，看到老板 PUA 替你生气。

> 【图片占位：截图问答功能演示，框选一段内容后 AI 给出分析回复】

### 3. 戳一戳互动

点击桌面上的小黑，它会有各种反应——摸摸头会咕噜咕噜，揉肚子会炸毛，夸它可爱会嘿嘿笑。而且连续戳它的话，反应会逐步升级，非常有趣。

> 【图片占位：戳一戳互动演示，展示不同表情GIF的切换效果】

### 4. 智能待办管理

直接在聊天中说「明天下午三点提醒我开会」，AI 会自动识别并创建待办事项，到时间弹窗提醒。对话即管理，零学习成本。

> 【图片占位：待办窗口截图，展示已创建的待办列表和提醒弹窗】

### 5. 主动关心你

小黑不是一个被动的工具。它会：
- **根据时间段主动打招呼**：早上说早安、中午问你吃了没、深夜劝你早点睡
- **随机碎碎念**：「刚才梦到自己变成一条咸鱼在晒太阳…」
- **陪伴里程碑**：陪伴满 7 天、30 天、100 天时会说一段特别的感言
- **久坐提醒 & 喝水提醒**：可自定义间隔时间，提醒方式可爱又不烦人

> 【图片占位：主动问候气泡截图 + 喝水提醒弹窗截图】

### 6. 喝水提醒——最卷的催喝水

这个喝水提醒我做得特别用心。弹窗会问你「该喝水啦！」，如果你点「不喝」，小黑会一步步变脸——从「要不再想想？」到「不许选这个！」再到「我会很伤心…」，表情也会从好奇变成怀疑、复杂、生气，直到躺平。最后还会说「啊我死了…快喝水！」按钮变成「复活」。

> 【图片占位：喝水提醒弹窗的多个阶段截图拼图，展示表情渐变过程】

## 二、角色系统——不只有小黑

### 内置角色

项目内置了**罗小黑**和**自嘲熊**两个角色，一键切换。每个角色都有完整的性格预设、专属表情包和互动文案。

> 【图片占位：设置面板的角色切换界面截图】

### 创建你自己的角色

这是我觉得最酷的部分：你可以**创建完全自定义的角色**。

**方式一（推荐，零代码）**：在设置中点击「创建新角色」，输入名字和性格描述，AI 会自动生成完整的角色配置——包括多套性格预设、各场景的回复风格、互动文案等。

**方式二（进阶）**：直接编辑 `characters/` 文件夹下的 JSON 文件，所有字段都有中文注释说明。

表情素材也可以自定义——放入 GIF 文件，在配置中映射表情名和文件名，AI 就会根据语境自动选择合适的表情。

> 【图片占位：角色创建器界面截图，展示填写名称和性格描述的表单】

## 三、快捷键一览

| 快捷键 | 功能 |
|--------|------|
| `Alt+F1` | 快捷聊天（在桌宠旁弹出迷你输入框） |

所有快捷键都可以在设置中自定义录制，支持 Ctrl/Shift/Alt 组合键。

## 四、手把手部署教程

### 环境准备

你需要先安装 **Node.js**（建议 16 以上版本）。

去 [Node.js 官网](https://nodejs.org/) 下载 LTS 版本安装即可。安装完成后打开终端验证：

```bash
node -v
npm -v
```

看到版本号就说明安装成功了。

> 【图片占位：终端显示 node -v 和 npm -v 输出的截图】

### 第一步：获取代码

打开终端，运行：

```bash
git clone https://github.com/kabaiye/AI-DeskPet.git
cd AI-DeskPet
```

或者直接去 GitHub 页面下载 ZIP 解压。

> 【图片占位：GitHub 仓库页面截图，标注 Code → Download ZIP 按钮位置】

### 第二步：安装依赖

在项目目录下运行：

```bash
npm install
```

等待安装完成即可，大概需要 1-2 分钟。

> 【图片占位：终端 npm install 完成的截图】

### 第三步：配置 API Key

AI-DeskPet 使用**智谱 AI** 的免费模型，你需要注册一个账号获取 API Key。

1. 访问 [智谱 AI 开放平台](https://open.bigmodel.cn/usercenter/proj-mgmt/apikeys)
2. 注册/登录后，在「项目管理 → API Keys」中创建一个 Key
3. 复制这个 Key

> 【图片占位：智谱 AI 平台的 API Key 管理页面截图，标注创建和复制按钮】

配置 Key 有两种方式（选一种即可）：

**方式一：在应用内设置（推荐）**

启动应用后，右键桌宠 → 打开设置 → 在「API Key 设置」中粘贴你的 Key。

**方式二：环境变量文件**

复制项目根目录的 `.env.example` 为 `.env`，填入你的 Key：

```
DASHSCOPE_API_KEY=你的API Key
```

### 第四步：启动！

```bash
npm start
```

你应该能看到小黑出现在桌面右下角，同时弹出主窗口显示「小黑已上线，准备陪伴你的每一天！」

> 【图片占位：首次启动后的桌面效果截图，小黑趴在桌面上 + 主窗口】

### 第五步（可选）：打包为安装包

如果你想把它打包成 EXE 安装包分享给朋友：

```bash
npm run build:win
```

打包完成后在 `dist` 目录下找到安装包。打包后的应用中 `characters/` 和 `assets/` 文件夹是外置的，用户可以直接添加新角色和素材，不需要重新打包。

> 【图片占位：dist 目录下生成的安装包文件截图】

## 五、进阶玩法

### 自定义角色 JSON 详解

每个角色的配置文件位于 `characters/` 目录下，结构清晰，所有字段都有 `_说明` 注释。核心字段包括：

- **`personality`**：角色核心性格 + 多套可切换预设
- **`style`**：不同场景（聊天/截图/提醒/碎碎念/戳一戳）的回复风格
- **`assets.emotions`**：表情名到 GIF 文件的映射
- **`interactions`**：戳一戳的触发语、升级反应、兜底回复
- **`fallbacks`**：AI 不可用时的预设回复
- **`waterReminderUI`**：喝水提醒的全套文案

举个例子，想让角色在被戳时说「别碰我的尾巴！」，只需要在 `interactions.pokeFallbacks` 中添加对应条目就行。

### 换接其他 AI 模型

项目底层用的是 OpenAI 兼容格式的 API 调用，理论上支持所有兼容接口。修改 `.env` 中的 `DASHSCOPE_BASE_URL` 和模型名称即可切换到其他服务商（如阿里云通义、月之暗面 Kimi 等）。

```
DASHSCOPE_BASE_URL=https://你的API地址/v1/
AI_TEXT_MODEL=你的文本模型名
AI_VISION_MODEL=你的视觉模型名
```

## 六、技术栈简介

给感兴趣的开发者同学介绍一下技术选型：

| 层级 | 技术 |
|------|------|
| 桌面运行时 | Electron |
| 主进程 | Node.js（CommonJS） |
| 渲染层 | 原生 HTML/CSS/JS（无框架，轻量） |
| AI 调用 | OpenAI SDK（兼容智谱/通义等） |
| 本地存储 | electron-store + JSON 文件 |
| 截图能力 | electron-screenshots |
| 打包工具 | electron-builder（NSIS 安装包） |

整个项目没有使用 React/Vue 等前端框架，代码结构简单直接，非常适合想学 Electron 开发的同学参考学习。
